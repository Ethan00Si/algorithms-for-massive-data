2.数据集:Data_Week3/

3.测试数据：
输入文件为图数据，第一行为节点数，之后每行表示一条边，节点编号从0开始。
输出文件第一行0表示没有哈密顿回路，1表示有哈密顿回路。如果有回路的话，第二行输出具体路径。
Example 1:
Input：
5
0 1
1 2
2 3
3 4
Output：
1
0 1 2 3 4

Example 2:
Input：
5
0 1
1 0
2 3
3 4
Output：
0
